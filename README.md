# Dot files

Config for

- Git
- Vim
- Bash
- Tmux
- Zsh

### Install

~~~bash
git clone https://github.com/Smirnov-O/dotfiles
cd dotfiles
chmod +x install.sh
./install.sh
vim +source~/.vimrc +PlugInstall
chsh -s $(which zsh)
~~~
